Introduction

C# (pronounced "C-sharp") is a modern, object-oriented programming language developed by Microsoft. It was first released in 2000 as part of Microsoft’s .NET initiative and has since evolved into one of the most popular languages for developing Windows applications, web applications, games (using Unity), and enterprise software.

C# is known for its versatility, strong typing, and ease of use, making it a go-to language for both beginners and experienced developers. The language combines the power of C++ with the simplicity of Visual Basic, and it’s designed to work well with the .NET framework.

Applications of C#:

  • Windows desktop applications
  • Web development (with ASP.NET)
  • Game development (especially with Unity)
  • Mobile app development (using Xamarin)
  • Cloud-based services (with Azure)

C# Syntax and Basics

Understanding C# starts with learning its basic syntax and how to declare variables, constants, and perform simple operations.

Data Types: C# is a statically-typed language, meaning you need to declare the type of data a variable will hold. Common data types include:

  • int – for whole numbers (e.g., int age = 25;)
  • double – for floating-point numbers (e.g., double price = 9.99;)
  • string – for text (e.g., string name = "Elden";)
  • bool – for true/false values (e.g., bool isActive = true;)

Variables and Constants: Variables are used to store data, and constants hold values that won’t change during the program's execution.

                
                    int level = 5; // A variable
                    const int maxHealth = 100; // A constant

Control Flow Statements

Control flow statements allow you to dictate the flow of your program by making decisions or repeating actions based on certain conditions.

if-else Statements: The if statement checks a condition, and if it's true, it executes the associated code. An optional else block can run code when the condition is false.


                if (health < 50) {
                    Console.WriteLine("Health is low!");
                    } 
                else {
                    Console.WriteLine("You're safe.");
                    }
switch Statements: Use a switch statement when you have multiple conditions to check. It is cleaner and more readable than using many if-else statements. 

                switch (weaponType) {
                    case "sword":
                        Console.WriteLine("You have a sword.");
                        break;
                    case "bow":
                        Console.WriteLine("You have a bow.");
                        break;
                    default:
                        Console.WriteLine("Unknown weapon.");
                        break;
                }

Loops: Loops help you run a block of code multiple times.

  • for loop: Repeat a task a certain number of times.
  • while loop: Repeat as long as a condition is true.

                for (int i = 0; i < 10; i++) {
                    Console.WriteLine(i);
                }

Object-Oriented Programming in C#

C# is an object-oriented programming (OOP) language, meaning that everything in C# is based on the concept of objects and classes.

Classes and Objects: A class is a blueprint for creating objects (instances). Objects represent real-world entities like characters in a game, cars, or even employees.


                class Player {
                    public string name;
                    public int level;
                
                    public void Attack() {
                        Console.WriteLine(name + " attacks!");
                    }
                }
                
                // Creating an object
                Player radahn = new Player();
                radahn.name = "Radahn";
                radahn.level = 50;
                radahn.Attack();

Inheritance allows a class to inherit fields and methods from another class, promoting code reuse.


                class Warrior : Player {
                    public void Defend() {
                        Console.WriteLine(name + " defends!");
                    }
                }

Polymorphism and Encapsulation: Polymorphism allows objects to take multiple forms, while encapsulation hides details to prevent outside interference.


Exception Handling

In C#, exceptions are used to handle errors gracefully. Instead of crashing the program, you can catch and manage these errors.

Try-catch Blocks: The try block contains code that might throw an exception, and the catch block handles the error.


                try {
                    int result = 10 / 0;
                } catch (DivideByZeroException ex) {
                    Console.WriteLine("Cannot divide by zero!");
                }

Throwing Exceptions: You can also throw exceptions manually when certain conditions occur.


                if (health < 0) {
                    throw new InvalidOperationException("Health cannot be negative.");
                }

C# Collections and LINQ

C# provides various collections to store and manage groups of related objects.

Arrays: Arrays hold a fixed number of elements of the same type.


                int[] scores = new int[5];

Lists: Lists are more flexible than arrays and can dynamically resize.


                List<string> inventory = new List<string>();
                    inventory.Add("Sword");
                    inventory.Add("Shield");

Dictionaries: Dictionaries store key-value pairs, allowing you to map one value to another.


                Dictionary<string, int> playerStats = new Dictionary<string, int>();
                playerStats.Add("Strength", 10);
                playerStats.Add("Dexterity", 8);

LINQ: Language-Integrated Query (LINQ) allows you to query collections in a more readable way.


                var highScores = scores.Where(score => score > 50);

File I/O in C#

C# makes it easy to read from and write to files using the System.IO namespace.

Reading a file:


                string text = File.ReadAllText("path_to_file.txt");
                Console.WriteLine(text);

Writing to a File:


                File.WriteAllText("path_to_file.txt", "Hello, Elden Ring!");

Advanced C# Features

C# includes advanced features like delegates, events, and asynchronous programming to enhance flexibility and performance.

Delegates: A delegate is a type that represents references to methods.


                public delegate void AttackDelegate();
                public class Warrior {
                    public void Attack() {
                        Console.WriteLine("Warrior attacks!");
                    }
                }

Asynchronous Programming: Asynchronous methods (async and await) improve performance by not blocking the main thread.


                public async Task FetchData() {
                    await Task.Delay(2000);
                    return 42;
                }
